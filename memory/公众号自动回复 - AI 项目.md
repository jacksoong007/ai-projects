## 配置步骤

### 1. 登录微信公众平台
访问：https://mp.weixin.qq.com

### 2. 进入自动回复设置
**左侧菜单：** 内容与互动 → 自动回复 → 关键词回复

### 3. 添加回复规则
点击「添加回复」，填写：

| 配置项 | 值 |
|--------|-----|
| **规则名称** | AI 项目资源包 |
| **关键词** | `AI 项目` |
| **匹配规则** | 包含关键词 |
| **回复内容** | 上方文案 |

### 4. 保存并测试
- 点击「保存」
- 用手机公众号发消息「AI 项目」测试
